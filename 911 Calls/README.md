<h2>Overview</h2>

<h2>Data Dictionary</h2>

| Variable  | Definition                        | Key                 |
| --------- | --------------------------------- | ------------------- |
| lat       | Latitude                          |                     |
| lng       | Longitude                         |                     |
| desc      | Description of the Emergency Call |                     |
| zip       | Zipcode                           |                     |
| title     | Title                             |                     |
| timeStamp | Time                              | YYYY-MM-DD HH:MM:SS |
| twp       | Township                          |                     |
| addr      | Address                           |                     |
| e         | Dummy Variable                    | Always 1            |
